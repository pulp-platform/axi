Module axi_serializer

Serialize all AXI transactions to a single ID (zero).

This module contains one queue with subordinate port IDs for the read direction and one for the write direction. These queues are used to reconstruct the ID of responses at the subordinate port. The depth of each queue is defined by MaxReadTxns and MaxWriteTxns, respectively.

Parameters

MaxReadTxns: int unsigned

Maximum number of in flight read transactions.

MaxWriteTxns: int unsigned

Maximum number of in flight write transactions.

IdWidth: int unsigned

AXI4+ATOP ID width.

axi_req_t: type

AXI4+ATOP request struct definition.

axi_rsp_t: type

AXI4+ATOP response struct definition.
