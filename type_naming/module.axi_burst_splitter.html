Module axi_burst_splitter

Split AXI4 bursts into single-beat transactions.

Limitations

  • This module does not support wrapping (axi_pkg::BURST_WRAP) bursts and responds to such bursts with slave error(s).
  • This module does not support atomic operations (ATOPs) and responds to ATOPs with a slave error. Place an axi_atop_filter before this module if upstream modules can generate ATOPs.
