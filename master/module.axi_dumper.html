Module axi_dumper

Simulation-Only dumper for AXI transactions

This module writes all handshaked AXI beats to a log file. To use in simulation,

ensure TARGET_SIMULATION is defined.

Parameters

BusName:

LogAW: bit

LogAR: bit

LogW: bit

LogB: bit

LogR: bit

axi_req_t: type

axi_resp_t: type

Ports

clk_i: input logic

rst_ni: input logic

axi_req_i: input axi_req_t

axi_resp_i: input axi_resp_t
